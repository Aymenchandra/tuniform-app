# Tuniform App

![Tuniform-app](./public/img/github-pict.png)

## Start project 
=> npm install

## Run project
=> npm run dev

## Important Notes
1) create .env
2) add DATABASE_URL="mysql://USERNAME:PASSWORD@HOST:PORT/DATABASE"
3) Run CMD => npx prisma migrate dev --name init
4) Run CMD => npx prisma generate
